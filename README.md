# Agent OS

## Installation

```bash
npm install @wrtnio/agent-os
yarn add @wrtnio/agent-os
pnpm install @wrtnio/agent-os
```

and install language model library

```bash
npm install openai
yarn add openai
pnpm install openai
```

## How to run benchmark?

```bash
npm install
npm pack
cd benchmark
npm install
npm run start
```
